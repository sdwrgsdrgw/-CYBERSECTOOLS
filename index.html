Network Scanner Guide

$ nmap --help

Nmap is the most powerful network scanner available.

$ Basic Scan

nmap -sS -sV [target]

This performs a SYN scan (-sS) and service version detection (-sV)

$ Advanced Scan

nmap -A -T4 [target]

Aggressive scan (-A) with timing template 4 (fast)

$ Port Range

nmap -p 1-1000 [target]

Scans ports 1 through 1000

$ OS Detection

nmap -O [target]

Attempts to identify the operating system

$ Script Scanning

nmap --script=[script-name] [target]

Runs NSE (Nmap Scripting Engine) scripts

$ Output to File

nmap -oN output.txt [target]

Saves results to output.txt

$ Legal Note

Only scan networks you own or have permission to scan.

Password Cracker Guide

$ john --help

John the Ripper is a powerful password cracking tool.

$ Basic Dictionary Attack

john --wordlist=[wordlist.txt] [hashes.txt]

Uses a wordlist to crack passwords

$ Brute Force Attack

john --incremental [hashes.txt]

Tries all possible character combinations

$ Rule-Based Attack

john --rules --wordlist=[wordlist.txt] [hashes.txt]

Applies transformation rules to wordlist

$ Show Cracked Passwords

john --show [hashes.txt]

Displays already cracked passwords

$ Hashcat Example

hashcat -m 0 -a 0 [hashes.txt] [wordlist.txt]

Uses hashcat for GPU-accelerated cracking

$ Legal Note

Only crack passwords you own or have permission to test.

Vulnerability Scanner Guide

$ nikto --help

Nikto is a web server vulnerability scanner.

$ Basic Scan

nikto -h [target]

Scans a target web server for common vulnerabilities

$ Scan Specific Port

nikto -h [target] -p [port]

Scans a specific port on the target

$ Scan with SSL

nikto -h [target] -ssl

Scans an SSL-enabled web server

$ Nessus Basics

1. Create new scan policy

2. Configure target IP range

3. Run scan and review results

4. Export report for analysis

$ OpenVAS Setup

1. sudo openvas-setup

2. Access web interface at https://127.0.0.1:9392

3. Create new task and run scan

$ Legal Note

Only scan systems you own or have permission to test.

WiFi Analyzer Guide

$ airodump-ng --help

Airodump-ng is part of the aircrack-ng suite for WiFi analysis.

$ Monitor Mode

airmon-ng start [interface]

Puts your wireless card in monitor mode

$ Capture Traffic

airodump-ng [interface]

Shows all nearby wireless networks and clients

$ Targeted Capture

airodump-ng -c [channel] --bssid [MAC] -w [file] [interface]

Captures packets from a specific network

$ Wireshark for WiFi

1. Capture in monitor mode

2. Use display filter "wlan"

3. Analyze 802.11 frames and management packets

$ Kismet Setup

1. sudo kismet

2. Configure source in web UI

3. View networks and clients in real-time

$ Legal Note

Only analyze networks you own or have permission to monitor.

Exploit Framework Guide

$ msfconsole

Metasploit Framework is the most popular exploit development platform.

$ Search for Exploits

search [keyword]

Searches available modules by keyword

$ Use an Exploit

use exploit/[path]

set RHOSTS [target]

set PAYLOAD [payload]

exploit

$ Meterpreter Basics

sysinfo - System information

shell - Get command shell

download/upload - File transfer

screenshot - Capture screen

$ Creating Payloads

msfvenom -p [payload] LHOST=[your_ip] LPORT=[port] -f [format] > output

Generates custom payloads for various platforms

$ Post Exploitation

run post/[module] - Run post modules

hashdump - Dump password hashes

persistence - Maintain access

$ Legal Note

Only use against systems you own or have permission to test.

Autopsy is a graphical interface for The Sleuth Kit.

3. Add image file

4. Analyze file system and artifacts

volatility -f [memory.dmp] --profile=[profile] pslist

volatility -f [memory.dmp] --profile=[profile] dumpfiles

$ Wireshark for Network Forensics

1. Open pcap file

2. Use display filters

3. Follow TCP streams

4. Export objects
